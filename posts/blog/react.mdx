1.React를 사용하는 이유가 무엇인가요? 장점은 무엇인가요?


1. 개발 속도가 빠르다.
Vanila javascript를 사용하여 개발하는 것은 리액트를 사용하는 것 보다 공수가 많이 든다.
Vanila javascript 아용하면 Dom을 조작하기가 번거롭기 때문이다.
element에 접근하여 이벤트 함수를 달아주거나 상태 변경 후 다시 랜더링 시키는 등의 작업을
리엑트에서는 자동으로 해주어 개발자가 비지니스 로직에 더 집 중 할 수 있게 한다. 

2. 재사용성이 좋다.
리액트는 컴포넌트 단위 개발을 제공한다.
컴포넌트에서는 props를 주입받아 상황에 따라 유연하게 
컴포넌트를 사용할 수 있게한다.
