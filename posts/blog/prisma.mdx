## Prisma
오픈소스 ORM
- Prisma Client: type-safe Query builder
- Prisma Migration
- Prisma Studio

## Prisma Shcema
datasouce, generator, model을 명시하는 스키마 파일
- datasource: database 연결 정보를 명시
- generator: prisma clinet를 생성하는 도구
- model: 어플리케이션 model 명시

## prisma generate
- `node_modules/.prisma/client`에 prisma client를 생성
- model 변경시 실행해 주어야 변경 사항이 반영됨
