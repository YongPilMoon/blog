## Action
Action은 reducer에서 store 상태변경을 결정하는 객체로 `type`, `payload` 속성을 갖습니다.
<br/>
### type
type은 일반적으로 string으로 정의되고 type에 따라 reducer에서 어떤 상태 변경 로직이 실행될지 결정됩니다.

<CodeBlock>
{`
function counter(state: any, action: Action) {
  switch (action.type) {
    case "INCREMENT":
      return state + 1;
    case "DECREMENT":
      return state - 1;
    default:
      return state;
  }
}
`}
</CodeBlock>
<br />
### payload
`playload`는 Redux에서 상태 변경에 필요한 정보를 전달하는 데 사용됩니다.
예를 들어 채팅 서비스 구현을 위해 Redux에 사용자가 참여한 채팅방 정보와 대화 내용을 저장했다고 가정해 봅시다.
대화 상태를 업데이트하기 위해선 사용자가 입력한 text와 어떤 채팅방에서 대화가 발생했는지를 알아야 합니다.
이때 `roomId`와 `text` 같은 정보를 `payload`로 전달할 수 있습니다.
